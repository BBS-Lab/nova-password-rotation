# Security Policy

## Supported versions

The latest stable release receives security updates.

## Reporting a vulnerability

If you discover a security vulnerability, please email **paris@big-boss-studio.com** instead of
using the public issue tracker.

- We aim to acknowledge your report within **5 business days**.
- Please do not publicly disclose the issue until a fix has been released.

We appreciate your help in keeping this package and its users safe.
